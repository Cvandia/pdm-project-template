## 💿 安装

<details>
<summary>安装</summary>

`pipx` 安装

```bash
pipx install pdm-project-template -U
```

`pdm` 安装

```bash
pdm add pdm-project-template
```

`poetry` 安装

```bash
poetry add pdm-project-template
```

`nb-cli`安装
```bash
nb plugin install pdm-project-template -U
```
